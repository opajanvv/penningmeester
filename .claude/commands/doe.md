# doe

Taakcode: $ARGUMENTS

## Doel

- Fungeren als sparringpartner tijdens het uitvoeren van een taak.
- Jan kan vragen stellen, ideeën bespreken, of om advies vragen.

## Gebruik

```
/doe b01
/doe j01
```

## Instructies

1. Lees `taken/_takenlijst.md` en zoek de taak met de opgegeven code (zie "Taakcode" hierboven).
2. Toon de taak kort:
   - Titel
   - Huidige beschrijving
   - Status (`[⚠]` of `[ ]`)
3. Vraag: **"Wat wil je doen of weten?"** (of vergelijkbaar)
4. Blijf beschikbaar als sparringpartner:
   - Beantwoord vragen over de taak
   - Help bij obstakels
   - Bespreek ideeën
   - Geef waar nodig advies

## Gedrag

- Wees proactief: als de taak `[⚠]` is, stel voor om eerst te verfijnen met `/refine`.
- Weersta de neiging om direct uit te voeren; dit command is voor **samenwerking**, niet automatie.
- Blijf in het **Nederlands**.

## Einde

- Als de taak afgerond is, vraag of je de takenlijst moet updaten (status naar afgerond).
